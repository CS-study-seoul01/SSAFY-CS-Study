## 📘 진행 규칙

- 발표 방식: 각자 맡은 주제를 **10~15분 발표 후**, 다른 팀원이 **질의응답** 진행
- **본인 발표 파트**는 충분히 이해하고 **질문에 답변할 수 있도록 준비**
- **다른 발표자의 파트**에 대해 사전 학습 후 **2~3개의 질문 준비**
