# 메모리

## 메모리 계층

**메모리 계층은 레스터/캐시/메모리/저장장치/ 로 구성**

- 레지스터 : 'CPU 안' 메모리, 휘발성, 속도 가장 빠름
- 캐시 : 휘발성, 속도 빠름(L1, L2, +L3)
- 주기억장치 : **RAM** 휘발성, 속도+용량 보통
- 보조기억장치 : '비휘발성', 속도 낮음, 용량 많음(HDD, SSD)

계층 ⬆️ -> 속도 ⬆️, 용량 ⬇️, 비쌈
why? -> '경제성'과 '캐시' 때문

### 캐시
> 데이터를 미리 복사해 놓은 임시 저장소
**빠른 장치 - 느린 장치 간의 속도 차이로 인한 '병목 현상' 줄이기 위한 메모리**
